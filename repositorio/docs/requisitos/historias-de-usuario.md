História 1 - Como cliente, quero criar uma conta para acessar o sistema.
     Como cliente, quero criar uma conta informando e-mail e senha, para conseguir acessar o sistema de forma segura.

História 2 - Como cliente, quero visualizar os materiais disponíveis para locação.
    Como cliente, quero uma interface limpa e intuitiva que exiba os produtos a disposição, demarcando nome, preço, descrição e disponibilidade.

História 3 - Como cliente, quero alugar o Kindle e acompanhar prazos.
    Como cliente, quero alugar um Kindle e acompanhar horários de devolução, para gerenciar minhas alocações de forma prática.

História 4 - Como cliente, quero reservar uma mesa no café.
    Como cliente, quero reservar uma mesa no café informando data, horário e quantidade de pessoas, para garantir disponibilidade antes de chegar ao local.

História 5 - Como administrador, quero gerenciar o estoque de Kindles e de produtos.
    Como administrador, quero gerenciar o estoque de Kindles e de produtos, permitindo cadastrar, atualizar quantidades, visualizar entradas e saídas

História 6 - Como cliente, quero fazer pedidos de café e outros produtos pelo aplicativo.
    Como usuário quero  efetuar pedidos de produtos pelo aplicativos, tendo uma interface simples e intuitiva de check out com informações como lista de itens, preço total e sugestões de compra, para conseguir adquirir produtos de forma simples.

História 7 - Como gerente, quero visualizar relatórios financeiros de uso e locação.
    Como gerente, quero visualizar relatórios financeiros de uso e locação, incluindo faturamento diário, semanal e mensal, para analisar o desempenho do negócio e tomar decisões estratégicas.

História 8 - Como administrador, quero gerenciar os usuários do sistema.
    Como administrador, quero gerenciar os usuários do sistema, podendo cadastrar, editar, bloquear, remover e definir permissões, para garantir um controle adequado de acesso e segurança na plataforma.
